Add WatchdogService and WatchdogAlarmReceiver declarations. Insert BEFORE the BootReceiver declaration (around line 347):

```xml
<!-- Watchdog service for HyperOS/MIUI background-kill recovery -->
<service
    android:name=".background.watchdog.WatchdogService"
    android:foregroundServiceType="dataSync"
    android:exported="false" />

<receiver
    android:name=".background.watchdog.WatchdogAlarmReceiver"
    android:exported="false" />
```

Key points per CONTEXT.md decisions:
- D-11: foregroundServiceType="dataSync" (not specialUse — avoids Play Store justification)
- D-12: FOREGROUND_SERVICE_DATA_SYNC permission already exists in manifest
- D-20, D-21: Package path is `.background.watchdog.WatchdogService` and `.background.watchdog.WatchdogAlarmReceiver`
- exported="false": These are internal components, not exposed to other apps

    - grep -n 'foregroundServiceType="dataSync"' app/src/main/AndroidManifest.xml returns match on WatchdogService line
