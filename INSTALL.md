# Install Weather VN

Go to the [Releases page](https://github.com/wikiepeidia/vn-weather-with-locationiq-and-nominatim/releases) and download the latest APK file ending in `_standard.apk`.

Install it on your Android device and you are done!

## Flavors

| Flavor | File suffix | Notes |
|--------|-------------|-------|
| Standard (recommended) | `_standard.apk` | All weather sources including LocationIQ + Nominatim for Vietnamese addresses |

Both flavors are signed with the same key, so you can switch between them without reinstalling.

## Staying up to date with Obtainium

[Obtainium](https://github.com/ImranR98/Obtainium) can track updates automatically:

1. Open Obtainium and tap **Add App**
2. Enter the URL: `https://github.com/wikiepeidia/vn-weather-with-locationiq-and-nominatim`
3. Tap **Add** — Obtainium will notify you when a new release is available
